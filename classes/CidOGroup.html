Class CidOGroup

Grupo da CID-O

Tem como base as especificações das colunas do arquivo CID-O-GRUPOS.CSV, que contém a descrição dos grupos de categorias da morfologia de neoplasias (CID-O).

See

cidOGroupsStream — para ler o arquivo CID-O-GRUPOS.CSV da CID-10

Example

Obtendo os dados do grupo

import { CidOGroup } from 'br-cid10-csv';

const row = 'M000;M000;Descrição;C00.-';
const record = new CidOGroup(row);

record.catFirst;    // 'M000'
record.catLast;     // 'M000'
record.description; // 'Descrição'
record.refer;       // 'C00.-'
